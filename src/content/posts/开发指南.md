### Astro(fuwari主题)的安装

1. 安装npm与git
2. *进入项目* `cd xxx`
3. 在项目目录运行 `npm create fuwari@latest`

### 初始配置

1. *进入项目* `cd xxx`
2. *启动开发服务器* `pnpm dev`
3. 打开浏览器看 http://localhost:4321
4. 通过配置文件 `src/config.ts` 自定义博客

### 部署站点至 GitHub Pages

请先按照官方文档操作

> [官方文档](https://docs.astro.build/zh-cn/getting-started/)

**初始化 git(如果还没有)**

- `git init`

**关联远程仓库**

- `git remote add origin https://github.com/yourusername/your-repo.git` (修改username和repo)
- `git branch -M main`

**推送代码**

- `git add .`
- `git commit -m "Initial commit"`
- `git push -u origin main --force`

**等待 GitHub Actions 自动部署，去仓库的 Actions 标签页看进度**
